# Analyse Zomato Bengaluru

## Description
Projet d’analyse exploratoire des restaurants de Bengaluru à partir des données Zomato.

## Structure du projet
- data/ : jeu de données
- notebooks/ : notebook Jupyter
- streamlit_app/ : application Streamlit
- rapport/ : rapport PDF

## Exécution
### Notebook
jupyter notebook

### Application Streamlit
streamlit run streamlit_app/app.py
